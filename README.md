# 프로그래머스 데브코스 final project : 수리 (Su Re)


## Introduction
프로젝트 설명 + 어떤 기능을 수행하는지 등 적으면 될것같아요


## Requirements
OS : AWS Linux 2  
Docker : v20.10.7  
Docker-compose : v2.4.1  
Nvidia Driver : 470.103.01    
CUDA : v11.4



## Build & Run
`npm run build`  
패키지 파일 생성


`docker-compose --build --d`  
Docker 이미지 생성, 컨테이너 생성 + 빌드 + 실행
